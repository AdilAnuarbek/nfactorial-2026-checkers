## 📖 О проекте

**Шашки Pro** - это современная веб-платформа для игры в шашки с красивым интерфейсом, умным ИИ и полной реализацией правил. Проект создан как демонстрация навыков full-stack разработки и может стать основой для реального коммерческого продукта.

### 🎨 Для кого это?

- **Любители шашек** - приятный интерфейс и плавная игра
- **Новички** - подсказки возможных ходов и разные уровни сложности ИИ
- **Разработчики** - чистый, масштабируемый код на современном стеке

### ✨ Почему это ценно?

В отличие от большинства онлайн-шашек, этот проект фокусируется на:
- **Пользовательском опыте** - красивый градиентный дизайн, плавные анимации
- **Умном ИИ** - три уровня сложности с оценкой позиций
- **Правильной реализации** - полная поддержка всех правил, включая обязательные взятия и дамки
- **Современных технологиях** - готов к масштабированию и добавлению новых фичей

## 📦 Установка и запуск

### Предварительные требования

- Node.js 18+ 
- npm или yarn

### Быстрый старт

```bash
# 1. Клонируйте репозиторий
git clone <your-repo-url>
cd nfactorial-2026-checkers

# 2. Установите зависимости
npm install

# 3. (Опционально) Supabase — скопируйте .env.example в .env.local
cp .env.example .env.local
# Заполните NEXT_PUBLIC_SUPABASE_URL и NEXT_PUBLIC_SUPABASE_ANON_KEY
# Выполните SQL из supabase/schema.sql в Supabase SQL Editor

# 4. Запустите dev-сервер
npm run dev

# 5. Откройте браузер
# http://localhost:3000
```

Без Supabase приложение полностью работает: игра, localStorage, история партий на устройстве.

### Команды

```bash
npm run dev          # Запуск в режиме разработки
npm run build        # Сборка для production
npm run start        # Запуск production сервера
npm run lint         # Проверка кода
```

## 🎯 Алгоритмы и логика

### Игровая логика

**Валидация ходов:**
- Проверка диагонального движения
- Расчет возможных взятий (включая цепочки)
- Определение обязательности взятий
- Превращение в дамку

**ИИ алгоритм:**
```
1. Получить все возможные ходы
2. Приоритет взятиям (обязательные правила)
3. Оценка позиции:
   - Взятия: +100 за каждую фигуру
   - Продвижение к дамке: +2 за каждый ряд
   - Центральная позиция: +3 за близость к центру
4. Выбор хода с максимальной оценкой
```
